# HoloPass — Landing Page (Front-End Design)

Global Solution 2026 · 1º Semestre · **Indústria Espacial** · FIAP

Landing page do HoloPass em **HTML + CSS puro**, com as 6 seções exigidas.

## Estrutura

```
front-end-landing/
├── index.html          # marcação (conteúdo)
├── css/
│   ├── reset.css       # reset CSS
│   └── style.css       # variáveis CSS, Flexbox, cores e responsivo
├── img/                # imagens SVG (todas com atributo alt no HTML)
│   ├── logo.svg
│   ├── hero-satellite.svg
│   ├── bracelet.svg
│   └── city-metro.svg
├── integrantes.txt
└── link_github.txt
```

## Como abrir

Abra `index.html` no navegador (duplo clique já funciona, pois é HTML/CSS puro).

## Requisitos atendidos

- **Menu de navegação funcional** (âncoras com rolagem suave).
- **6 seções:** Problema, Tecnologia, Objetivos, Público-alvo, Benefícios e
  Aplicação no dia a dia.
- **Rodapé padronizado** com primeiro nome e RM.
- **Imagens com `alt`**, ícones, **Google Fonts** (Orbitron + Inter).
- **Reset CSS**, **variáveis CSS** e layout em **Flexbox**.
- Links estilizados com `hover`, paleta padronizada e textos curtos.
